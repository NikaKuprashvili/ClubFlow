# ⚽ ClubFlow: Advanced Sports Analytics & Management System

**ClubFlow** არის მაღალი დონის სპორტული ანალიტიკისა და კლუბის მენეჯმენტის სისტემა (SaaS პროდუქტი), რომელიც შექმნილია საფეხბურთო კლუბების სპორტული დირექტორებისა და ანალიტიკოსებისთვის. აპლიკაცია უზრუნველყოფს გუნდის ფინანსურ აუდიტს, ფეხბურთელების ეფექტურობის ღრმა მათემატიკურ ანალიზს და მოწინავე სკაუტინგს რეალურ სტატისტიკურ მონაცემებზე (FBref) დაყრდნობით.

---

## 🛠️ ტექნოლოგიური სტეკი (Tech Stack)

პროექტი მკაცრადაა დაყოფილი დამოუკიდებელ მოდულებად (Modular Architecture), რათა უზრუნველყოფილ იქნას კოდის სისუფთავე და გუნდური მუშაობის ეფექტურობა.

* **Core Logic:** Python 3.11+, Object-Oriented Programming (OOP)
* **Database & ORM:** SQLite + **SQLAlchemy (Advanced Database Tech - ნასწავლი პროგრამის მიღმა)**
* **User Interface (UI):** PyQt6
* **Advanced UI Component:** **HTML5 / CSS3 ინტეგრაცია PyQt-ში (Advanced Frontend Tech - ნასწავლი პროგრამის მიღმა)**
* **Data Visualization:** Matplotlib (დინამიკური გრაფიკები და რადარ-ჩარტები)
* **Reporting:** ReportLab (ავტომატური PDF რეპორტების გენერაცია)

📋 ფუნქციონალური გეგმა და გვერდების აღწერა (5 Pages Blueprint)
1. Auth & Onboarding Page (ავტორიზაცია)
რა კეთდება: სპორტული დირექტორების უსაფრთხო ავტორიზაციის სისტემა.

ანალიტიკური ლოგიკა: სისტემაში შესვლისას, ბაზიდან იტვირთება კონკრეტული დირექტორის user_id, რის მიხედვითაც პროგრამა ახდენს იდენტიფიკაციას და ტვირთავს მხოლოდ მისი კლუბის მონაცემებს.

2. Executive Dashboard (მთავარი ანალიტიკური პანელი)
რა კეთდება: კლუბის ზოგადი "ჯანმრთელობისა" და ფინანსური მდგომარეობის შეჯამება (KPIs).

ანალიტიკური კომპონენტები:

Squad Balance Metric: ალგორითმი, რომელიც აანალიზებს გუნდის ასაკობრივ სტრუქტურას და პოზიციურ დეფიციტს.

Performance vs Wage: Matplotlib გრაფიკი, რომელიც აჩვენებს კლუბის სპორტული შედეგების კორელაციას სახელფასო ფონდთან.

სმარტ-შეტყობინებები: სისტემური გაფრთხილებები (მაგ. "მარცხენა მცველების საშუალო რეიტინგი დაბალია ლიგის საშუალოზე 12%-ით").

3. Squad Performance Manager (შემადგენლობის ღრმა ანალიზი)
რა კეთდება: მიმდინარე ფეხბურთელების ეფექტურობის აუდიტი.

ანალიტიკური კომპონენტები (UI ინტეგრაცია HTML/CSS-ით):

Value-for-Money Index (VFM): მათემატიკური ფორმულა, რომელიც ფეხბურთელის სათამაშო სტატისტიკას ადარებს მის ხელფასს.

Fatigue & Injury Predictor: ნათამაშები წუთებისა და ასაკის მიხედვით ტრავმის რისკის პროგნოზირება.

Contract Expiry Matrix: ვიზუალური დაფა, რომელიც ფერებით (HTML/CSS ბარათები) მიანიშნებს, თუ რომელი მოთამაშის კონტრაქტი იწურება მალე.

4. Advanced Scouting Market & Radar Comparison (სკაუტინგი)
რა კეთდება: ახალი ტალანტების ძებნა და გუნდის გაძლიერების სიმულაცია.

ანალიტიკური კომპონენტები:

Similarity Search Algorith: სპორტული დირექტორი ირჩევს სასურველ ვარსკვლავს და სისტემა fbref_market.csv-დან ავტომატურად პოულობს იაფფასიან ანალოგებს მსგავსი სტატისტიკური პროფილით.

Radar Chart Comparison: Matplotlib-ის მრავალკუთხა რადარ-ჩარტი, სადაც ვიზუალურად ერთმანეთს გადაედება ორი ფეხბურთელის სათამაშო კომპონენტები (დრიბლინგი, პრესინგი, პასის სიზუსტე).

5. Financial Simulator & PDF Report Generator (ფინანსური სიმულატორი)
რა კეთდება: ბიუჯეტის კონტროლი და სტრატეგიული დაგეგმვა.

ანალიტიკური კომპონენტები:

Transfer Sandbox: ფეხბურთელის შეძენის სიმულატორი. სისტემა წინასწარ ითვლის, ხომ არ დაარღვევს კლუბი ფინანსურ ფეირ-პლეის (FFP) მომავალი 3 წლის ჭრილში.

Automated Executive Summary: ღილაკზე დაჭერით გენერირდება სრულფასოვანი ანალიტიკური PDF რეპორტი, სადაც შედის გუნდის სუსტი წერტილები და რეკომენდირებული ტრანსფერები.

📈 დამოუკიდებელი და მოწინავე ტექნოლოგიების დასაბუთება
Backend - SQLAlchemy (ORM): ნაცვლად სტანდარტული და მარტივი sqlite3-ისა, პროექტში გამოყენებულია ინდუსტრიული სტანდარტის Object-Relational Mapping (ORM) ტექნოლოგია. მონაცემთა ბაზის ცხრილები რეალიზებულია პითონის სუფთა OOP კლასებად, რაც უზრუნველყოფს მონაცემების მანიპულაციას SQL თხოვნების ხელით წერის გარეშე.

Frontend - HTML5 & CSS3 PyQt-ში: ინტერფეისის მაქსიმალური დახვეწილობისთვის, ფეხბურთელების ანალიტიკური ბარათები და პროგრესის ზოლები აწყობილია HTML/CSS არქიტექტურით და ჩაშენებულია PyQt კომპონენტებში, რაც სცილდება სტანდარტული PyQt ვიჯეტების დიზაინის ჩარჩოებს.
