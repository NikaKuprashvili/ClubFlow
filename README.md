# ⚽ ClubFlow — Advanced Sports Analytics & Club Management System

## 📖 პროექტის მიმოხილვა

**ClubFlow** წარმოადგენს სპორტული ანალიტიკისა და საფეხბურთო კლუბების მართვის თანამედროვე პროგრამულ სისტემას (SaaS კონცეფცია), რომელიც შექმნილია სპორტული დირექტორების, სკაუტების, ანალიტიკოსებისა და მენეჯერული პერსონალისთვის.

პროექტის მთავარი მიზანია საფეხბურთო კლუბის სპორტული და ფინანსური გადაწყვეტილებების მიღების პროცესის მონაცემებზე დაფუძნებული ანალიზით მხარდაჭერა. სისტემა აერთიანებს ფეხბურთელების ინდივიდუალური ეფექტურობის შეფასებას, შემადგენლობის სტრუქტურულ ანალიზს, ტრანსფერული ბაზრის კვლევას, ფინანსური რისკების სიმულაციასა და ავტომატური ანალიტიკური ანგარიშების გენერაციას.

ტრადიციული კლუბური მართვისგან განსხვავებით, სადაც გადაწყვეტილებები ხშირად მხოლოდ ადამიანურ გამოცდილებას ან ინტუიციას ეყრდნობა, ClubFlow ცდილობს ამ პროცესში რაოდენობრივი ანალიზისა და ვიზუალური მონაცემების ინტეგრირებას, რათა სპორტულ დირექტორს უფრო ინფორმირებული და ობიექტური გადაწყვეტილებების მიღების საშუალება მიეცეს.

სისტემა მუშაობს ცენტრალიზებული მონაცემთა ბაზის, მათემატიკური ანალიტიკური მოდულებისა და ინტერაქტიული ვიზუალიზაციების საფუძველზე, რაც საშუალებას იძლევა ერთ სივრცეში მოხდეს როგორც მიმდინარე მდგომარეობის შეფასება, ასევე მომავალი სცენარების მოდელირება.

# 🛠️ ტექნოლოგიური სტეკი (Tech Stack)

პროექტი აგებულია მოდულური არქიტექტურის პრინციპით, რაც უზრუნველყოფს კოდის მასშტაბირებადობას, გამართულ სტრუქტურასა და გუნდის წევრებს შორის პასუხისმგებლობების ეფექტურ გადანაწილებას.

### Core Logic

* Python 3.11+
* Object-Oriented Programming (OOP)

ბიზნეს-ლოგიკა, ანალიტიკური გამოთვლები და სისტემის ძირითადი ალგორითმები რეალიზებულია ობიექტზე ორიენტირებული პროგრამირების პრინციპებით.

### Database Layer

* SQLite
* SQLAlchemy ORM

მონაცემთა ბაზასთან ურთიერთობა ხორციელდება SQLAlchemy ORM-ის საშუალებით, რაც ცხრილებს პითონის კლასებად გარდაქმნის და მნიშვნელოვნად ამარტივებს მონაცემების დამუშავებას.

### User Interface

* PyQt6

მომხმარებლის გრაფიკული ინტერფეისი აგებულია PyQt6 ფრეიმვორკის გამოყენებით.

### Advanced UI Technologies

* HTML5
* CSS3
* PyQt Integration

გარკვეული ანალიტიკური კომპონენტები და ვიზუალური ელემენტები რეალიზებულია HTML/CSS ტექნოლოგიებით და ინტეგრირებულია PyQt გარემოში.

### Data Visualization

* Matplotlib

გამოიყენება:

* KPI გრაფიკებისათვის
* ტრენდების ანალიზისთვის
* რადარ-ჩარტებისთვის
* ფინანსური ვიზუალიზაციებისთვის

### Reporting Engine

* ReportLab

სისტემის მიერ გენერირებული ანალიტიკური ანგარიშების PDF ფორმატში ექსპორტისთვის.

# 🏗️ სისტემის არქიტექტურა

# 📈 მოწინავე ტექნოლოგიების გამოყენების დასაბუთება

## SQLAlchemy ORM

პროექტში გამოყენებულია SQLAlchemy ORM, რომელიც წარმოადგენს ინდუსტრიულ სტანდარტს მონაცემთა ბაზებთან მუშაობისთვის.

მისი გამოყენება უზრუნველყოფს:

* სუფთა არქიტექტურას;
* OOP მიდგომის სრულ ინტეგრაციას;
* მონაცემთა ბაზის უფრო მარტივ მართვას;
* კოდის მასშტაბირებადობას.

## HTML5 & CSS3 ინტეგრაცია PyQt-ში

სტანდარტული PyQt ვიჯეტების ვიზუალური შესაძლებლობები შეზღუდულია.

ამ პრობლემის გადასაჭრელად ანალიტიკური ბარათები, პროგრესის ინდიკატორები და სტატუსების ვიზუალიზაცია რეალიზებულია HTML/CSS ტექნოლოგიებით.

ეს მიდგომა უზრუნველყოფს:

* თანამედროვე დიზაინს;
* უკეთეს UX/UI გამოცდილებას;
* მოქნილ ვიზუალურ კონტროლს;
* პროფესიონალურ ანალიტიკურ დაფებს.
